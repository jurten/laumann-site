Accesibilidad

En Laumann & Asociados trabajamos para ofrecer un sitio web claro, legible y fácil de utilizar para todas las personas, especialmente para quienes necesitan una experiencia digital simple y comprensible.

Nuestro compromiso

Nos esforzamos por brindar una navegación simple, con textos legibles, estructura clara y contenido comprensible, para que cada persona pueda acceder a la información con comodidad.

Medidas adoptadas

  • Uso de tipografías legibles y jerarquías visuales claras.
  • Tamaños de texto y contrastes pensados para facilitar la lectura.
  • Estructura simple, con secciones ordenadas y navegación comprensible.
  • Botones y enlaces visibles, con textos descriptivos.
  • Redacción clara y directa para reducir la complejidad innecesaria.

Mejora continua

La accesibilidad es un proceso permanente. Revisamos y mejoramos de forma continua el sitio para ofrecer una experiencia más clara, estable y útil.

Asistencia

Si tiene dificultades para visualizar contenidos, completar formularios o navegar este sitio, puede comunicarse con nosotros. Estaremos disponibles para brindarle una alternativa de contacto y la ayuda necesaria.
